# Garden Diary Mapper (GitHub Pages)

A browser-based version of your Python mapper.

## Files
- [index.html](index.html): neon UI + mapping logic (runs fully in browser)

## How to use
1. Open the web page.
2. Upload **Source workbook**.
3. Upload **Mapping workbook**.
4. Click **Generate Output**.
5. Downloaded file will be the updated mapping workbook.

## GitHub Pages deploy
1. Push this folder to a GitHub repository.
2. In repo settings: **Pages** → **Build and deployment**.
3. Source: **Deploy from a branch**.
4. Branch: `main` (or your branch), folder `/ (root)`.
5. Save and open the Pages URL.

No backend required.
